HTML Tagovi




  • <html>: Ovaj tag označava početak i kraj HTML dokumenta.
  • <head>: Tag za zaglavlje HTML dokumenta.
  • <body>: Tag koji označava telo HTML dokumenta.
  • <title>: Tag koji definiše naslov dokumenta.
  • <h1>: Tag za najveći naslov.
  • <p>: Tag za paragraf.
  • <font-face>: Ovaj tag nije standardni HTML tag i verovatno je greška. Koristi se CSS za stilizaciju fonta.
  • <mark>: Tag za isticanje teksta.
  • <u>: Tag za podvlačenje teksta.
  • <b>: Tag za podebljanje teksta.
  • <i>: Tag za kurzivni tekst.
  • <table>: Tag za kreiranje tabele. 
    <table>
    <tr>
        <th>Ime</th>
        <th>Prezime</th>
    </tr>
    <tr>
        <td>Ismet </td>
        <td>Fetic</td>
    </tr>
</table>
  • <ul>: Tag za nenumere liste. 
    <ul>
    <li>Predmet 1</li>
    <li>Predmet 2</li>
</ul>
  • <ol>: Tag za numerisane liste. 
    <ol>
    <li>Predmet 1</li>
    <li>Predmet 2</li>
</ol>
  • <img>: Tag za slike. 
    <img src="slikamojekite.jpg" alt="deskripcija slike">
